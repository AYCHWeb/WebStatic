Open Web 0.9 gelanceerd

Open Web 0.9 gelanceerd

Door Levente Bokor
Geschreven op 2016-03-04

Iedereen die in de creatieve industrie werkt kent het fenomeen. Je kunt de mooiste dingen voor je klanten maken, maar als het op een website voor jezelf aankomt, is niets goed genoeg. Althans, halverwege het uitwerken van het ene ontwerp zit je alweer na te denken over het volgende.

Zo kan het ook bestaan dat een website voor een full-service internetbureau als Open Web lang op zich heeft moeten laten wachten. Omdat we perfectionisten zijn, maar ook omdat de technologie in de internetwereld razendsnel verandert. We willen het neusje van de zalm gebruiken, een robuuste solution voor onszelf opzetten.

Toch breekt nood soms wet. We moeten nu eenmaal een website hebben. Er zijn vele schetsen aan voorafgegaan en twee volledige ontwerpen zijn de prullenmand in verdwenen... maar versie 0.9 van de website van Open Web is een feit.

Versie 0.9, want wij zijn Hippo-professionals en de website die u nu leest, draait op een tamelijk ouderwets PHP-systeem. Zeer binnenkort gaan onze technici aan de slag om de volledige website om te bouwen naar Hippo 10, een zeer krachtig en complex Java CMS.

Wij wensen u veel plezier met versie 0.9 van onze website. Wij gaan snel aan de slag met versie 1.0. Wilt u zien wat wij allemaal met onder andere Hippo, Angular, React, Java, iOS, Android, Docker en DevOps kunnen? Neem dan eens  contact met ons op.

06.04.2016
